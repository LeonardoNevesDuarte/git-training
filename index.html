Git Training

  • Steps to Use Git
    • git init => To initialize local empty repo
    • git add *.* => To add files
    • git commit -m "commit message" => Commit changes with a specific message
    • git branch -M main => Move/Rename branch to specifi name
    • git remote add origin https://github.com/LeonardoNevesDuarte/git-training.git => Adding local repo to remote repo on Git
    • git push -u origin main => Push changes to remote repo
